IDirectFBDisplayLayer   SetSrcColorKey

Set the source color key.

SetSrcColorKey (
  IDirectFBDisplayLayer   * thiz,
  u8   r,
  u8   g,
  u8   b
);

If a pixel of the layer matches this color the underlying pixel is visible at this point.
